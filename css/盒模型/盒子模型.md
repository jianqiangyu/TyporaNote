# CSS盒子模型

CSS盒模型是CSS规范的一个模块，它定义了一个长方形的盒子，每个盒子拥有各自的内边距和外边距，并根据视觉格式化模型来对元素进行布局。

当对一个文档布局的时候，浏览器会根据盒子模型将所有元素表示为一个举行盒子，css决定这些盒子的大小，位置以及属性（颜色，背景，边框尺寸）
